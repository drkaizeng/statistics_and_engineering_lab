# Data structures

## K-d tree (in prep)
For a dataset with $N$ records and $k$ real-valued attribute for each record, the time complexity for searching the best matches of a query record is $kN \, \text{log} N$.


!!! note "References"
Friedman, J.H., Bentley, J.L. and Finkel, R.A., 1977. An algorithm for finding best matches in logarithmic expected time. ACM Transactions on Mathematical Software (TOMS), 3(3), pp.209-226.
